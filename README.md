## 🛡️ Featured Project — StackSentry
 
> **Automated web application security assessment, AI-powered remediation, and auto-fix.**
 
[![PyPI](https://img.shields.io/badge/pypi-v1.0.0-orange)](https://pypi.org/project/stacksentry/)
[![Tests](https://img.shields.io/badge/tests-316%20passing-brightgreen)](https://github.com/stacksentry/stacksentry)
[![Python](https://img.shields.io/badge/python-3.10%2B-blue)](https://pypi.org/project/stacksentry/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
 
StackSentry scans your full web application stack — Flask/Django app, Nginx/Apache, Docker containers, and Linux host — across **24+ security checks**, assigns a security grade (A–F), maps findings to **OWASP Top 10**, and generates AI-powered fix scripts that can be auto-applied via SSH or directly to your config files.
 
```bash
stacksentry --target https://your-app.com --mode full --patch --fix \
  --ssh-host your-server-ip --ssh-user root \
  --dockerfile ./Dockerfile --compose-file ./docker-compose.yml
```
 
**What it does in one command:**
- Runs 24+ security checks across 4 layers (App → Web Server → Container → Host)
- Generates a professional PDF report with OWASP Top 5 mapping & prioritised hardening plan
- Creates AI-powered fix scripts tailored to your stack (Anthropic API integration)
- Auto-applies fixes via SSH or directly to config files
- Tracks security posture drift between scans
